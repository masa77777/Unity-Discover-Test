## プライバシーポリシー

### 処理するユーザーデータの内容
ユーザーID、プロフィール、アバター

### そのデータの処理方法
研究開発アプリの検証において、MR機能を検証するために必要となります。

### そのデータの処理目的
ユーザーIDは、複数セッション間でユーザーを識別するため。  
プロフィール情報（ユーザーネーム、プロフィール写真）をアプリ内で該当ユーザーまたはその他のユーザーに表示するため
アバターは、ユーザーのアバターを表示するため。
社内での研究開発目的でのみ利用し、本免情報はいかなる第3者にも提供しません。
